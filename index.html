Image Grabber JS

Image Grabber JS is a package and API for fetching random images from reddit. The package and API can be used for fetching any images from the internet, and providing an imgur link. I have also made a package that wraps it, that allows you to use the API, from an npm package. Image Grabber JS is commonly used for image commands in discord bots, and a few other things. With a fully functioning website, you will deem this helpful.

JupitrBot

JupitrBot is a discord bot written in JavaScript, and Node JS. It has an extremely complex folder structure, that makes the project much cleaner. This bot is highly advanced, with many different commands and features. The bot uses many different APIs and it's own Database to store data.

Player Controller Script (Unity)

This is a simple player controller script for Unity. It comes with features such as walking around, mouselook, and jumping. The script is made in C#, and works because of the Physics Rigidbody component in Unity. The purpose of this script was to give people learning a foundation to be able to make games in Unity.
